About VIKI

VIKI was created to teach individuals how to play the piano and learn musical notes without purchasing a keyboard. VIKI was constructed using the following...

Tone.js

VIKI incorportated the Tone.js framework for creating interactive music in the browser. It provides advanced scheduling capabilities along with synths and effects while you play they keyboard on the Home page.

YouTube

YouTube, a video sharing website, was integrated using an API. YouTube was key in order for VIKI to display keyboard-how-to videos.

Firebase

Firebase was utilized to capture data from the form shown on the homepage and display the captured data within a table on the Leaderboard page.

Bootstrap

Bootstrap is a free and open-source front-end library for designing websites and web applications. Bootstrap was used throughout VIKI.

Animate.css

Animate.css is a just-add-water css animation. Great for emphasis, home pages, sliders, and general just-add-water-awesomeness. VIKI had some fun with this within the jumbotron.
